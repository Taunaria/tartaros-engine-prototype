# Tartaros Prototype (Fantasy Demo)

This repository contains a playable Godot prototype, not the later engine.

Current prototype loop:
- Warrior-only real-time combat
- 6 handcrafted 32x32 levels
- 1 reward chest per level
- linear progression from village to abyss
- simple boss ending the demo

Controls:
- `WASD` or arrow keys: move
- `Space`: attack
- `E`: open chest

Important:
- no engine abstraction yet
- no procgen
- no inventory screen
- temporary blocky placeholder art by design
